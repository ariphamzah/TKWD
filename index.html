************************************************************************************************************************************************************************

address merupakan tag untuk mendefinisikan kontak informasi pembuat atau pemilik dari sebuah dokumen atau web

contoh :

Jl. Babakan Karees No.04

article merupakan tag untuk mendefinisikan sebuah article, digunakan untuk memberi mark up sebuah independent content, artinya element itu dapat berdiri sendiri sebagai sebuah konten utuh yang tidak terikat dengan konten lain (element lain).

contoh :

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Facere voluptate totam aperiam. Explicabo provident nisi possimus rem quam tempora quaerat veniam sapiente velit laborum cum, illo ipsum, eius quo magnam?
